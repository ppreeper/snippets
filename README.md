# snippets
snippet library for vscode

## Linux
```bash
cd /home/{user}/.config/Code/User
rm -rf snippets
git clone https://github.com/ppreeper/snippets
```

## Windows
```powershell
cd C:\Users\{user}\AppData\Roaming\Code\User
Remove-Item -Recurse -Force snippets
git clone https://github.com/ppreeper/snippets
```
